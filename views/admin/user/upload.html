Importer depuis un fichier CSV

Sélectionner votre fichier CSV pour importer les utilisateurs décris dedans.
Le séparateur de champs à l'utiliser est la virgule `,`

Format de fichier attendu :

  • login Précisez la valeur pour modifier l'utilisateur ou laissez le champs vide pour créer l'utilisateur
  • categorie Valeur obligatoire
  • prénom
  • nom
  • adresse email Valeur obligatoire
